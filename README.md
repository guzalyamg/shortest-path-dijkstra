

# Shortest Path Finder — Dijkstra's Algorithm

A Python implementation of Dijkstra's shortest path algorithm 
with graphical visualization of the graph and the calculated path.

## Tech Stack
- Python
- Graph algorithms
- Graphical visualization

## Features
- Implements Dijkstra's algorithm from scratch
- Visualizes nodes, edges, and the shortest path graphically
- Demonstrates understanding of data structures and algorithm design

## How to Run
1. Make sure Python is installed
2. Run the file:
   python shortest_path_finder.py

## Project Context
Built as part of Computer Science coursework at 
Webster University Tashkent.
